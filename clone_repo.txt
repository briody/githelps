Clone Your Repo (Like for another machine)

1. git clone https://github.com/usename/repo_name.git
2. git add (new file to add)
3. git commit -m "message"
4. git push

if you get an error like:
Permission denied (publickey).
fatal: The remote end hung up unexpectedly

See this page: https://help.github.com/articles/generating-ssh-keys

It will help you set up a public key and show how to add it on github.
